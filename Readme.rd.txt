1|| ¿Qué es un clase "sealed"?

Es una clase a la cual se limita su entorno de herencia, se la usa con la palabra reservada sealed. Es decir,evita que se pueda heredar de esa clase a otra.

2|| ¿Cuáles son los operadores de una expresión relacional?

Los operadores relacionales son:

< "menor que"
> "mayor que"
== "mayor que"
!= "No igual a"
<= "menor que o igual a"
>= "mayor que o igual a"
